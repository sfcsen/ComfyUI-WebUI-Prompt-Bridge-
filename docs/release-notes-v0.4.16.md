# ComfyUI WebUI Prompt Bridge v0.4.16

本次更新主要修复 Bridge 面板的布局抖动和旧工作流按钮遮挡问题，建议所有 v0.4.x 用户更新。

## 更新内容

- 点击 `恢复尺寸` 后，LoRA 区域不再来回抖动。
- 拖动提示词标签区和 LoRA 区后，底部不再留下大片空白。
- LoRA 卡片区高度更稳定，重启后不再异常变长。
- 旧工作流里的右侧开关不再被 Bridge 节点遮挡，按钮可以正常点击。
- 增加自动回归检查，覆盖旧工作流、布局拖拽、设置保存和真实生图。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.16`。
