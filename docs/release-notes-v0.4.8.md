# ComfyUI WebUI Prompt Bridge v0.4.8

本次更新集中修复 Bridge 主节点的布局保存问题。

## 更新内容

- 刷新页面、重启 ComfyUI 或切换工作流后，会保留用户手动调整过的 Bridge 节点宽高。
- 打开工作流时不再先出现异常高度再回弹，节点布局会更稳定。
- 修复部分工作流重新打开后，Bridge 节点界面下移、顶部留白或提示词区域被挤走的问题。
- 只有节点尺寸数据明显异常时，才会自动恢复到安全尺寸。
- 优化提示词导入和错误提示，遇到问题时更容易看懂。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.8`。
