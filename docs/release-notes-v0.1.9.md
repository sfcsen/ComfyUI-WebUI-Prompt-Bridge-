# ComfyUI WebUI Prompt Bridge v0.1.9

这是 v0.1.8 之后的补丁版，主要修正 WebUI 配套扩展检测和提示文案。

## 主要更新

- 已经安装 Prompt All in One / TagComplete 的用户会看到 `扩展已存在`，不需要重复下载。
- 只有 WebUI 缺少扩展时，才需要使用 `补齐缺失扩展`。
- 没有 WebUI 的用户仍然可以使用 `下载本地数据包`，把词库下载到本节点自己的 `data` 目录。
- 如果前端已更新但后端还没重启，补齐扩展接口可能返回 404 / 405。现在会明确提示重启 ComfyUI。

## 怎么选择

- 已有 WebUI 和两个扩展：直接 `一键接入 WebUI`。
- 有 WebUI 但缺扩展：点 `补齐缺失扩展`。
- 没有 WebUI：点 `使用内置数据` 或 `下载本地数据包`。

## 注意

更新后请重启 ComfyUI，再刷新浏览器页面。
