# ComfyUI WebUI Prompt Bridge v0.4.4

本次补丁重点修复节点界面稳定性和操作手感。推荐所有 v0.4.x 用户更新，尤其是遇到过切换 ComfyUI 布局尺寸后节点闪烁的用户。

## 修复内容

- 修复在 ComfyUI `设置 - 布局/显示` 中切换布局尺寸后，WebUI Prompt Bridge 主节点可能不停闪烁的问题。
- 节点尺寸、区域高度和布局缓存现在会跳过无变化写入，减少重复重排。
- 切换 `默认 / 紧凑 / 宽松 / 正向聚焦 / 极简 LoRA` 布局后，节点会更稳定地停在目标尺寸。

## 操作体验

- 顶部按钮、尺寸快捷和布局预设按钮更容易点击。
- 拖拽分隔条和右下角整体缩放手柄增加拖动中的视觉反馈。
- 布局预设按钮会根据侧栏宽度自动换行，窄侧栏下不再挤成一排。
- 键盘焦点和触控操作反馈更清楚。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

如果之前已经遇到节点闪烁，建议在刷新后重新切换一次 Bridge 节点里的布局预设，或点击 `恢复尺寸` 让节点重新写入稳定尺寸。

本版本保留现有 `[tool.comfy]` 注册表元数据，包版本为 `0.4.4`。推送到 `main` 后，仓库中的 `Publish to Comfy Registry` GitHub Action 会触发 Comfy Registry 发布。
