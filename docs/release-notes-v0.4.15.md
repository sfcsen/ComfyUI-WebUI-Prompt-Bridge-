# ComfyUI WebUI Prompt Bridge v0.4.15

本次更新修复旧示范工作流和 Bridge 节点操作稳定性问题，建议所有 v0.4.x 用户更新。

## 更新内容

- 旧版示范工作流重新可用，打开后会自动兼容旧字段顺序。
- 修复点击节点内 `Generate` 后，界面可能变窄、空白或错位的问题。
- 鼠标在 Bridge 节点内滚轮时，不再误缩放 ComfyUI 画布。
- 修复生成后拖拽调整节点尺寸可能导致页面无响应的问题。
- 保留用户自定义节点宽高，拖拽和刷新后的内部布局更稳定。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.15`。
