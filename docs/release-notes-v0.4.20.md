# ComfyUI WebUI Prompt Bridge v0.4.20

本次更新新增侧边栏生成参数接管，并修复宽高、Seed 与实际采样器参数不一致的问题。需要从 Bridge 面板直接控制出图尺寸、Steps 或 Seed 的用户建议更新。

## 更新内容

- 侧边栏新增 `生成参数`：可分别开关接管出图宽高、Steps、Seed 和 Seed 模式。
- `读取` 会按当前工作流实际连接读取参数；有上游常量时读取上游，没有上游时读取节点内置值。
- `应用` 和 `Generate` 会把启用的参数写回实际生效位置，避免面板数值和 KSampler 使用值不一致。
- 修复出图宽高没有正确映射到真实输出尺寸的问题。
- 修复 Seed 显示值和 KSampler 实际 Seed 不一致的问题。

## 验证

- 已用真实 ComfyUI 生图验证上游宽高和内置宽高，输出 PNG 尺寸与面板设置一致。
- 已用真实 ComfyUI 生图验证内置 Seed 和上游 Seed，两组不同 Seed 生成结果不同。
- 已通过前端语法检查和后端 Python 编译检查。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.20`。
