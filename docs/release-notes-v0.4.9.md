# ComfyUI WebUI Prompt Bridge v0.4.9

本次更新修复 Bridge 主节点在不同主题和不同 ComfyUI 前端结构下的默认布局兼容问题。

## 更新内容

- 新拖出的 `WebUI Prompt Bridge` 会直接使用宽版默认布局，不再变成窄高条。
- 面板会更稳定地铺满节点主体，减少右侧裸露、下方空白和控件被挤压的问题。
- 如果旧工作流里已经保存了异常的窄高节点尺寸，打开后会自动恢复到安全宽版尺寸。
- 默认、紧凑、宽松、正向聚焦和极简 LoRA 布局都重新校准了高度分配，LoRA / Extra Networks 区域会更好地利用底部空间。
- 加强对暗色主题、亮色主题、主题插件强制宽高、旧版和新版 DOMWidget 包裹结构的兼容。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.9`。
