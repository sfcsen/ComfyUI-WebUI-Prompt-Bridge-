# ComfyUI WebUI Prompt Bridge v0.4.5

本次更新重点修复节点界面错位和提示词导入失败。推荐所有 v0.4.x 用户更新，尤其是遇到过“只剩侧栏”“提示词区域消失”“恢复尺寸没用”的用户。

## 修复内容

- 修复部分工程重新打开后，Bridge 主节点可能只剩侧栏、提示词区域消失或出现大片空白的问题。
- `恢复尺寸` 和 `默认尺寸` 会同时清理异常布局缓存，界面错位时不需要删除节点重连。
- 打开旧工程时会自动修复异常保存的 Bridge 节点尺寸，降低再次打开又错位的概率。
- 修复导入提示词时可能提示 `Cross-origin requests are not allowed` 的问题。
- 优化导入、设置、AI、词库和资源下载等错误提示，不再直接显示原始 JSON。
- 补强图片输入节点和本地缓存处理，减少浏览器缓存异常造成的界面中断。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

如果节点界面已经错位，刷新后点击 Bridge 节点顶部的 `恢复尺寸`，或右侧尺寸区域的 `默认尺寸`。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.5`。
