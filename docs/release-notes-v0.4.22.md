# ComfyUI WebUI Prompt Bridge v0.4.22

本次热修复撤销远景摘要对节点尺寸的自动调整，避免缩放恢复后 Bridge 面板布局错乱。

## 更新内容

- 低于 68% 缩放时仍显示简洁摘要，但不再改变 Bridge 节点尺寸。
- 不再移动连线端点，不再触发 ComfyUI 的节点尺寸与 DOM 面板重算。
- 回到正常缩放时直接显示原面板内容和活动标签，不执行尺寸恢复。
- 工作流保存尺寸和画布上的节点尺寸在整个缩放过程中保持一致。

## 验证

- 验证 100%、55% 和 32% 缩放切换。
- 验证连续跨越 68% 阈值时节点尺寸不变。
- 验证低缩放保存、恢复内容和 Bridge 控制台无错误。

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.22`。
