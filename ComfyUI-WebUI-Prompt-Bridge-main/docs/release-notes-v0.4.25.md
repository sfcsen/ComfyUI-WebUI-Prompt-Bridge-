# ComfyUI WebUI Prompt Bridge v0.4.25

这次更新主要解决两个使用问题：主节点功能越来越多后不容易找到常用入口，以及高级模块看起来“可以用”、实际接线时才发现缺模型或缺输入。更新后，日常编辑更紧凑，高级功能也会先告诉你当前环境究竟准备到了哪一步。

## 主界面更清楚

- 正向提示词和反向提示词改为页签切换，同一时间只展示当前编辑区域。
- LoRA 区把搜索、分页、排序和批量操作重新分组；没有勾选 LoRA 时不再显示无用的批量按钮。
- 设置页和布局工具减少重复入口，中文名称与按钮用途更统一。
- 窄节点会自动使用单列布局，常用拖拽和尺寸恢复入口仍然保留。

## 提示词市场不再只像“小节点专用”

提示词市场现在明确说明：下载的词条会进入 Bridge 的自定义词库，可供主节点以及正向、负向提示词小节点使用。

市场分类也补充了更适合出图的方向，包括：

- 插画、动漫、写实、概念设计等画风
- 电影镜头、摄影构图和景别
- 光影、色彩、材质与氛围
- 质量增强、细节修复和常用负面词

导入后仍可在 `设置 -> 编辑词库` 中统一管理或清空市场词条。

## 高级模块先检查，再构建

ADetailer、ControlNet、SAM/Inpaint 和放大修复现在会显示真实准备状态：

- `可用`：所需节点包和关键模型已经找到。
- `部分可用`：Bridge 可以创建接入口，但第三方节点或主工作流输入仍需手动连接。
- `缺少资产`：缺少关键模型时会直接禁用构建按钮，并告诉你缺什么。
- `仅审计`：区域 LoRA 只统计使用情况，不会误导为已经实现区域隔离。

点击一键构建后，状态栏会列出新增节点数、自动连线数以及仍需接入的 IMAGE、VAE、MASK 或主采样链路。构建结果不合适时，可直接点击 `还原上次构建`。

## 不再假装第三方节点已经接好

- SAM 模块只创建 Mask/Inpaint 接入节点，不会猜测你安装的是哪一种 SAM Loader 或分割节点。
- Ultimate/Tiled 模式只保存当前参数，不会创建看起来像分块放大、实际却没有接入第三方插件的替代节点。
- ControlNet 会分别检查预处理器和控制模型；只安装 Canny/OpenPose 预处理器但没有 ControlNet 模型时，会明确显示缺少模型。

## 升级说明

- 原有主节点、小节点和工作流继续兼容，不需要重新创建节点。
- 更新后请重启 ComfyUI，并在浏览器中按 `Ctrl+F5` 强制刷新。
- 本版本同步发布到 GitHub 与 ComfyUI Registry，包版本为 `0.4.25`。
