# ComfyUI WebUI Prompt Bridge v0.4.6

本次更新继续修复 Bridge 主节点在重启、刷新或切换工作流后界面被挤到下方的问题。

## 修复内容

- 修复部分用户重新打开工作流后，Bridge 节点顶部出现大片空白、提示词面板被挤到下方的问题。
- 节点界面会更稳定地贴在节点顶部，切换多个工作流时不需要反复手动拖回布局。
- 保存工作流时会写入 Bridge 当前真实节点尺寸，降低异常尺寸被再次保存的概率。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.6`。
