# ComfyUI WebUI Prompt Bridge v0.2.3

这一版重点增强提示词词库、提示词编辑和 AI 翻译。

## 新增功能

- **提示词市场**：可以打开免费提示词来源，并一键导入公开词库，导入后自动分类。
- **重新导入**：已经导入过的来源会显示导入状态，也可以重新导入，方便误删后恢复。
- **全屏提示词编辑器**：可以在设置里集中新增、编辑、删除提示词，并按分类和来源筛选。
- **AI 接口设置**：支持 OpenAI 兼容接口，例如 SiliconFlow、DeepSeek 等。
- **检测模型**：填写 Base URL 和 API Key 后，可以读取上游真实模型 ID，点击即可填入模型名。
- **AI 翻译**：设置里选择 `AI 翻译` 后，可以用配置的 AI 接口把中文提示词翻译成英文 tag。

## 体验优化

- AI 翻译时，输入框会显示 `AI 正在翻译...` 并高亮，用户能看出功能已经触发。
- 自动补全会显示中文解释；本地词库不足时，可以开启联网补全中文解释。
- 提示词市场增加 `当前 WebUI / 本地词库` 导入入口，避免 WebUI 词库没有显示或本地未下载时无处补齐。
- SiliconFlow 推荐模型 ID 为 `deepseek-ai/DeepSeek-V4-Flash`；如果误填 `deepseek-v4-flash`，节点会自动修正。

## 升级提示

更新后请重启 ComfyUI，再刷新浏览器页面。

AI 接口的 API Key 只保存在本机 `config.local.json`，不会写入工作流。
