# ComfyUI WebUI Prompt Bridge v0.4.12

本次更新集中修复粉丝反馈的节点放置、工作流切换布局和 WebUI 接入卡住问题。

## 更新内容

- 修复部分旧版 ComfyUI 中，新建 `WebUI Prompt Bridge` 节点时无法放置，按 `Esc` 后画布也可能卡住的问题。
- 修复切换工作流后，Bridge 节点偶发恢复成默认显示、下方出现大片空白的问题。
- 保留用户手动摆好的节点尺寸，不再用频繁恢复默认来处理布局异常。
- 修复一键接入 WebUI 可能卡在 `8/9` 的问题；如果接入超时，按钮会恢复，可以重新点击。
- 优化 LoRA 浏览：没有缩略图时不再破图，多级文件夹点父目录也能看到子目录里的 LoRA。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.12`。
