# ComfyUI WebUI Prompt Bridge v0.4.14

本次更新继续修复 Bridge 节点布局稳定性问题，建议所有 v0.4.x 用户更新。

## 更新内容

- 修复刷新网页、加载旧工作流或重新新建节点后，LoRA 区下方出现大片空白的问题。
- 修复旧高度缓存和已保存节点尺寸冲突导致的底部抖动。
- 保留用户自定义的节点宽高，同时让 LoRA 区在有明显空白时自动填满。
- 优化底部间距和高度容差，刷新和拖拽后布局更稳定。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.14`。
