# ComfyUI WebUI Prompt Bridge v0.2.6

这是一个面向提示词面板布局的轻量更新。

## 新增内容

- **正向撰写优先**：放大正向 Prompt 输入区和正向标签区，并折叠反向提示词细节，适合集中写正向提示词。
- **隐藏 LoRA 最小化**：折叠 LoRA 区并缩小节点，适合只需要快速写提示词、不想让 LoRA 卡片占用空间的场景。
- **快捷布局按钮**：面板右侧新增 `默认`、`紧凑`、`宽松`、`正向`、`极简` 五个按钮，可以直接切换布局。

## Comfy Registry

本版本将 `pyproject.toml` 版本更新为 `0.2.6`，并保留现有 `[tool.comfy]` 注册元数据。推送到 `main` 后，仓库中的 `Publish to Comfy Registry` GitHub Action 会触发 Comfy Registry 发布。

## 验证

- 已在本地 ComfyUI `http://127.0.0.1:8188/` 打开运行副本验证。
- 已确认五个布局按钮可见。
- 已确认 `正向` 和 `极简` 两个新增预设可以切换并显示对应状态提示。

## 升级提示

更新后请重启 ComfyUI，并强制刷新浏览器页面，确保前端扩展脚本重新加载。
