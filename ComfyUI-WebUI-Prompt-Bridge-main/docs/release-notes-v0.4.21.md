# ComfyUI WebUI Prompt Bridge v0.4.21

本次更新让 Bridge 更紧凑、更易浏览，同时加强本地配置、文件路径和图片处理安全。

## 更新内容

- 默认使用紧凑布局，侧边栏按模型、生成、图生图和模块分区。
- 画布缩放低于 68% 时临时显示不超过 `920×320` 的摘要节点，连线端点同步收拢。
- 放大后恢复原始节点尺寸、活动标签和面板内容；低缩放保存仍写入原始尺寸。
- 加强 API Key、本地配置、请求来源、AI 地址和本地文件路径保护。
- 限制图片体积、像素数、动画帧数和响应大小，增强 Mask 与预览图处理安全。
- 优化图生图入口、图片编辑和提示词 Tag 操作。

## 验证

- 前端语法检查、Python 编译检查、`git diff --check` 和完整兼容测试。
- 真实 ComfyUI 下检查 100%、55%、32% 缩放及大尺寸旧工作流节点。
- 核对旧工作流尺寸、生成参数行为、版本、更新日志和 Registry 元数据兼容性。

## 升级说明

更新后请重启 ComfyUI，并强制刷新浏览器页面。

本版本会同步发布到 ComfyUI Registry，包版本为 `0.4.21`。
