# Notices

ComfyUI WebUI Prompt Bridge is an independent ComfyUI custom node that recreates a WebUI-like prompt editing workflow for ComfyUI.

This project can optionally read local data from the following WebUI extensions when the user configures a local WebUI path:

- `sd-webui-prompt-all-in-one` by Physton, MIT License.
- `a1111-sd-webui-tagcomplete` by Dominik Reh, MIT License.

The project also targets compatibility with workflows inspired by AUTOMATIC1111 Stable Diffusion WebUI. AUTOMATIC1111 Stable Diffusion WebUI is licensed under AGPL-3.0.

No model, LoRA, checkpoint, VAE, upscaler, ControlNet, detector, or other large AI weight file is bundled in this repository. Users are responsible for installing models and checking their individual licenses.
