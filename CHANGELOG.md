# 更新日志

## 0.2.9 - 2026-06-06

- 修复新添加 `WebUIPromptBridge` 节点时会自动套用已保存的 `宽松` 布局，导致节点第一次出现就被拉得很长的问题。
- 调整 DOM widget 默认高度计算，避免 ComfyUI 布局在首次添加节点后继续把默认高度从 `1040` 撑高。
- 保留手动布局预设按钮行为：用户仍可点击 `宽松`、`紧凑`、`正向`、`极简` 来主动改变节点尺寸。

## 0.2.8 - 2026-06-05

- 重做 Prompt Bridge 节点内布局：正向提示词固定在顶部，LoRA / Extra Networks 区域固定贴合底部，节点缩放后不再出现上方或下方大片空白。
- 放开节点宽高限制，右下角拖动可以把 UI 拉成长条或宽胖形态，内部面板会跟随节点尺寸重新分配空间。
- LoRA 区域增加明显边框和上下分隔拖拽条，可以把 LoRA 卡片区压缩到很小，也可以拉开浏览。
- 新增 LoRA 卡片分页控件，支持 32 / 48 / 72 / 96 / 全部；默认仍显示全部 LoRA。
- 优化 `默认`、`紧凑`、`宽松`、`正向`、`极简` 五种布局预设，让 LoRA 区在极简状态下仍保留可拖动的小条。

## 0.2.7 - 2026-06-05

- 修复 WebUI 接入后看起来卡在 `8/9` 的误导：状态会显示具体缺少的 WebUI 资源，例如 ControlNet，不再把可选目录缺失误认为接入失败。
- 优化 WebUI 接入刷新流程：LoRA 列表默认走轻量模式并带超时，避免大量 LoRA metadata 扫描导致按钮一直不结束。
- LoRA 列表刷新失败或超时时会保留原列表，并提示“LoRA 扫描较慢”，不会把卡片清空后仍提示刷新成功。
- 修复 Extra Networks / LoRA 区域下方出现大片空白的问题：普通节点里 LoRA 区按内容收缩，超过高度后内部滚动；快速添加 LoRA 的覆盖层模式仍可大面积浏览。
- LoRA 预览图改为懒加载，预览不存在时自动回退到 `NO PREVIEW`，不再阻塞 WebUI 接入流程。

## 0.2.6 - 2026-06-03

- 新增两个布局预设：`正向撰写优先` 会放大正向 Prompt 和正向标签区，并折叠反向提示词细节；`隐藏 LoRA 最小化` 会折叠 LoRA 区并缩小节点，适合只写提示词。
- 面板右侧新增 5 个快捷布局按钮：`默认`、`紧凑`、`宽松`、`正向`、`极简`，不用打开设置页也可以快速切换节点布局。
- 后端设置白名单同步支持新的 `positive_focus` 和 `minimal_lora`，保存工作流或重启 ComfyUI 后布局选择可以稳定保留。

## 0.2.5 - 2026-06-01

- 修复反向提示词区域里 `收藏列表`、`扩展模型` 打不开或刚切过去又跳回反向词的问题。
- `扩展模型` 里新增 Embedding 分类，会读取 ComfyUI 的 embeddings 目录，常用 embedding 可以直接点进提示词。
- 修复 WebUI 接入路径异常时设置页显示安装失败的问题；现在会给出更稳的检测结果，不会因为一个无效路径把整个状态接口打崩。
- 修复联网翻译开启后输入中文、点击 `英` 没反应的问题，并补齐实际运行需要的依赖声明。
- 优化 Prompt All in One 区域的高度、空列表提示和自动补全弹窗，内容不会再被挤到看不见。
- 修复设置里调大字体或换字体后，分类标签卡片中文字互相遮挡的问题；卡片高度现在会跟随字号自动调整。

## 0.2.4 - 2026-05-29

- 修复 LoRA “匹配成功但实际没生效”的误导：现在会区分“找到 LoRA 文件”和“LoRA 已进入采样器模型链路”。
- 明确支持内置 LoRA 加载：不需要额外连接 LoRA Loader，但必须把 `WebUIPromptBridge` 的 `model` 输出接到采样器实际使用的模型链路。
- 如果提示词里有 `<lora:name:weight>`，但本节点 `model` 输出没有接出去，前端会显示警告并阻止提交，避免用户白跑。
- 修复异步 LoRA metadata 检测把警告状态覆盖回“匹配 sdxl/noob”的问题，避免再次误判。
- 后端增加 LoRA 加载日志和 `lora_info` 警告：实际应用、上游已加载、缺失 LoRA、以及 model 输出未接都会有更清楚的信息。

## 0.2.3 - 2026-05-28

- 新增提示词市场：可以打开免费提示词来源，并一键导入公开词库，导入后自动分类；已导入来源会显示状态，也支持重新导入，方便误删后恢复。
- 新增全屏提示词编辑器：可以在设置里集中新增、编辑、删除提示词，支持分类、来源筛选，并能看到主页面/WebUI 词库内容。
- 增强 WebUI / 本地词库导入：提示词市场增加“当前 WebUI / 本地词库”入口，避免 WebUI 词库未显示或本地未下载时无处补齐。
- 增强中文补全解释：自动补全会优先读取本地中文词库，不足时可联网补齐 TagComplete 中文解释；`kamisa` 等英文输入也能显示中文说明。
- 新增 AI 接口设置：支持 OpenAI 兼容接口，可配置 Base URL、模型、API Key 和系统提示，API Key 只保存在本机 `config.local.json`。
- 新增模型检测：AI 设置里可以从上游 `/models` 读取真实模型 ID，点击即可填入；SiliconFlow 下会识别 `deepseek-ai/DeepSeek-V4-Flash`，也兼容用户输入的 `deepseek-v4-flash` 简写。
- 新增 AI 翻译来源：设置里可选择 `AI 翻译`，中文提示词会通过配置的 AI 接口翻译成英文 tag；翻译中会在输入框显示“AI 正在翻译...”并高亮提示。
- 优化联网翻译/补全设置：设置页可分别选择翻译来源、联网翻译和联网补全中文解释，方便没有完整本地词库的用户使用。

## 0.2.2 - 2026-05-27

- 修复右侧 `CLIP` 数字框被清空后生成时报 `Failed to convert an input value to a FLOAT value` 的问题；现在空值会自动回到 `1`。
- 加强工作流保存兜底：保存时会把异常的 LoRA 默认 CLIP 权重修正为数字，旧工作流或误操作不容易再保存出空 FLOAT。
- 后端也增加兜底：即使外部工作流传入空值或非数字值，也会按 `1.0` 处理。
- 修复设置里的字体大小不明显生效的问题：按钮、输入框、提示词标签、LoRA 卡片等主要文字现在会一起跟随字体大小变化，默认字体大小为 `12px`。
- 批量添加选中的 LoRA 到正向提示词后，会自动清空选择状态，绿色选中框和勾选框不会继续残留。
- 补充用户说明：明确 v0.2.0 已修复 LoRA 接入误报、刷新后提示词错位/不能保存、自定义标签增删改、多选拖拽等反馈，并写清楚多个提示词标签如何一起拖动。

## 0.2.1 - 2026-05-27

- 修复右侧 `CLIP` 数字框被清空后生成时报 `Failed to convert an input value to a FLOAT value` 的问题；现在空值会自动回到 `1`。
- 修复设置里的字体大小不明显生效的问题：按钮、输入框、提示词标签、LoRA 卡片等主要文字现在会一起跟随字体大小变化。

## 0.2.0 - 2026-05-27

- 修复 WebUI 接入后的 LoRA 误报缺失问题：现在会同步 WebUI/配置里的 LoRA 路径，并支持 `.lnk` 快捷方式、文件名、路径名和部分 metadata 别名匹配。
- 修复刷新页面后 Prompt / Negative prompt / CLIP 等字段错位的问题，减少“正向词跑到反向词里”和提示词无法保存的情况。
- 优化 LoRA 卡片库：入口改为 `快速添加 LoRA`，支持勾选多个 LoRA 后一次添加到正向提示词，并显示添加成功提示。
- 优化 LoRA 面板布局：折叠后只保留一条紧凑栏，拖动分隔线时不再把 LoRA 区压成一坨或撑出大片空白。
- 优化提示词输入：底部输入框更明显，按 Enter 添加时默认走翻译；输入过程中不再刚打一个字就自动翻译。
- 新增自定义提示词标签管理，可以新增、修改、删除自己的标签内容。
- 新增多选拖拽提示词标签：按住 Ctrl 多选后可以一起移动。
- 修正 LoRA 权重编辑：现在会正确修改 `<lora:name:weight>` 里的权重，而不是把整段 LoRA 标签当普通提示词加权。
- 已实测从浏览器工作流切换并生成成功：`waiANINSFWPONYXL_v60.safetensors` 整合 XL 模型、分体 `Anima/Qwen` 模型，以及从 Anima 切回 XL 后再次生成。

## 0.1.9 - 2026-05-27

- 优化 WebUI 配套扩展入口：已经安装 Prompt All in One / TagComplete 的用户会看到“扩展已存在”，不再提示重复下载。
- 修正前后端版本不一致时的报错说明：如果补齐扩展接口返回 404 / 405，会提示重启 ComfyUI 加载新的后端接口。
- 一键接入 WebUI 会同步返回配套扩展检测结果，方便用户判断是“直接接入”还是“补齐缺失项”。

## 0.1.8 - 2026-05-27

- 新增无 WebUI 独立模式：没有本机 WebUI 时也会提供内置基础 tag 分类、自动补全、常用反向词和简单中英映射。
- 新增首次启动向导：可选择连接 WebUI、使用内置数据或导入本地词库。
- 新增集中设置面板：管理数据来源、翻译来源、布局尺寸、Tag 显示方式和 LoRA 卡片大小。
- 新增词库导入接口，支持从 JSON / CSV / TSV 导入自定义 tag 映射。
- 新增配套数据补齐：已有 WebUI 和扩展时无需下载；WebUI 缺扩展时可只补齐缺失项；没有 WebUI 时可下载到本节点 data 目录直接使用。

- 修复 Negative prompt 折叠后仍然占用大量纵向空间的问题。
- 修复提示词 tag 编辑工具条在底部或右侧滚动条附近被遮挡的问题。
- 补充说明已有操作：双击提示词 tag 可以隐藏/禁用该 tag，再次双击可以恢复；被禁用的 tag 不会参与生成。
